# Suricata Network Traffic Lab

This repository contains a click‑by‑click lab guide for installing and running Suricata IDS on Ubuntu and using it to monitor and analyse network traffic.

## Contents

- **suricata_network_traffic_lab.docx** – step‑by‑step Word guide with installation instructions, log inspection, and custom rule creation.

## Summary

The lab walks you through:

1. Updating your system and adding the OISF `suricata-stable` PPA.
2. Installing Suricata from pre‑built packages.
3. Starting Suricata in IDS mode on your network interface.
4. Viewing and analysing events in the `eve.json` log file.
5. Creating a local detection rule and testing it by sending ping traffic.

Refer to the Word document for the full procedure and screenshots.
